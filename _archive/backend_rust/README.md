# Errant Fox Backend (Standalone)

Это выделенный бэкенд проекта Errant Fox. Он отвечает за хранение данных, обработку видео и логику HEMA-аналитики.

## Структура папок
*   `/server`: Исходный код на Rust (основная логика).
*   `/protobuf`: Определения протоколов связи (общие для сервера и клиента).
*   `/docs`: Документация по API и архитектуре.
*   `noodl_bridge.js`: Скрипт для интеграции в Noodl.

## Как запустить

### Способ 1: Напрямую через Rust (Рекомендуется для разработки)
1.  Установите [Rust](https://rustup.rs/).
2.  Установите **FFmpeg** (необходим для обработки видео).
3.  Перейдите в папку `server`:
    ```bash
    cd server
    ```
4.  Запустите сервер:
    ```bash
    cargo run --release
    ```
    Сервер будет доступен по адресу `ws://localhost:8095`.

### Способ 2: Через Docker
Если у вас установлен Docker, просто запустите:
```bash
docker-compose up --build
```

## Интеграция с UI (Noodl)
Используйте файл `noodl_bridge.js` внутри вашего проекта Noodl для отправки команд серверу. Подробности см. в `docs/API.md`.
